# 项目概述

## 项目目标

在一个自包含的 HTML 可视化大屏上，集中展示全国所有开通地铁的城市客流数据，使用户能够直观了解各城市地铁运营规模与历史趋势。

## 覆盖范围

- **总覆盖城市**：50 个
- **含客流数据城市**：34 个（数据来源于 MetroDB.org）
- **无客流数据城市**：16 个（MetroDB.org 标记为"无数据"，大屏上对应显示为"暂无数据"）
- **地图底图**：中国行政区划 GeoJSON，支持本地 `assets/china.json` → 远程 CDN → 城市列表表格三级降级策略

## 实时性说明

**本项目不是实时系统。** 所有数据均通过爬虫脚本一次性采集，不提供自动刷新、定时更新或推送能力。展示的是采集时刻的历史快照，不代表当前实际运营状况。

## v1.1 已完成能力

- [x] 从 MetroMan.cn 批量采集 50 个城市的地铁线路图/规划图（PNG 格式）
- [x] 从 MetroDB.org 采集 34 个城市的客流统计数据（JSON 格式）
- [x] 基于客流 JSON 数据生成年度趋势图与全国对比图（PNG 格式）
- [x] 将客流数据内嵌至 dashboard.html，使用 ECharts 实现浏览器端可视化大屏
- [x] 中国地图底图加载，支持本地/CDN/降级三种模式
- [x] 城市搜索与筛选功能
- [x] 响应式布局，适配不同屏幕尺寸
- [x] 无客流数据的 16 个城市明确标注"暂无数据"
- [x] `daily_ridership_wan = 0` 视为缺失值，不作为真实零值展示
- [x] 项目目录结构优化：城市目录收归 `cities/`，脚本分类至 `scrapers/` 和 `scripts/`

## 能力边界

以下能力在 v1.1 中 **不支持**，如有需要请参见 [ROADMAP.md](ROADMAP.md) 了解后续规划：

| 不支持项 | 说明 |
|----------|------|
| 实时数据 | 无 WebSocket、无轮询，数据为一次性采集快照 |
| 自动更新 | 无定时任务、无 CI/CD 触发的自动重采集 |
| 站点级数据 | 仅展示城市级聚合客流，不提供单站点进出站量 |
| 多版本对比 | 不支持选择不同日期的数据进行横向对比 |
| 后端服务 | dashboard.html 为纯前端自包含文件，无需后端服务器 |
| CI/CD 流水线 | 无持续集成/持续部署流程 |
| 用户认证 | 无登录、权限控制或数据隔离 |
| 数据导出 | 不支持 CSV/Excel 等格式的数据导出 |
