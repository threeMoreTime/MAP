# 爬取地铁线路图与规划图

> 本文档记录从 MetroMan.cn 爬取全国 50 个城市地铁线路图和规划图的完整方案。

---

## 数据源

**网站**：[MetroMan.cn](https://www.metroman.cn)

MetroMan.cn 提供各城市地铁线路图和规划图的在线浏览，底层使用 OpenSeadragon 深度缩放查看器。可直接访问的静态 PNG 图片遵循以下 URL 模式：

```
https://www.metroman.cn/assets/img/metro/{类型}/routemap_{城市代码}_cn.png
```

| 参数 | 值 | 说明 |
|------|----|------|
| `{类型}` | `map` | 运营线路图 |
| `{类型}` | `plan` | 规划线路图 |
| `{城市代码}` | 如 `bj`、`sh` | 城市拼音缩写（小写） |

示例：
- 运营线路图：`https://www.metroman.cn/assets/img/metro/map/routemap_bj_cn.png`
- 规划线路图：`https://www.metroman.cn/assets/img/metro/plan/routemap_bj_cn.png`

> **注意**：URL 中的城市代码使用拼音缩写（如北京为 `bj`），而非完整拼音。脚本内通过 `CITY_CODE_MAP` 字典自动映射。

## 脚本文件

**根目录脚本**：`scrape_subway_route_map.py`

该脚本为最终版本，包含：
- 50 个城市完整列表与中文名映射
- `CITY_CODE_MAP` 城市拼音 → 缩写映射（如 `beijing` → `bj`）
- SSL 证书验证跳过（MetroMan 站点 SSL 配置不标准）
- 占位图自动过滤（< 2KB 跳过）
- 8 并发线程下载
- 日志输出到 `scrapers/scrape_log.txt`

**历史脚本**：`scrapers/scrape_all_cities.py`（早期版本，功能已被根目录脚本替代）

## 占位图过滤

部分城市缺少规划图，MetroMan 返回小于 2KB 的占位 PNG。脚本通过 `MIN_FILE_SIZE = 2048` 阈值自动过滤：

| 情况 | 文件大小 | 处理 |
|------|----------|------|
| 正常线路图 | 50KB - 1.5MB | 保留 |
| 占位图（无规划） | ~1KB | 跳过不保存 |

实际爬取中，以下 9 个城市的规划图为占位图（无数据）：
- 佛山、哈尔滨、呼和浩特、金华、高雄、兰州、南通、绍兴、芜湖

- 网络可访问 www.metroman.cn

## 配套脚本

图片爬取完成后，可继续运行以下脚本补充数据：

| 脚本 | 数据源 | 输出 | 说明 |
|------|--------|------|------|
| `scrapers/scrape_metrodb.py` | metrodb.org | `cities/{city}/{city}_stats.json` | 客流统计数据 |
| `scrapers/scrape_metrodb_data.sh` | metrodb.org | 同上 | Bash 版本（功能相同） |
| `scrapers/generate_charts.py` | 本地 stats JSON | `cities/{city}/{city}_yearly_trend.png` | 年度趋势图表 |

推荐执行顺序：

```bash
python scrape_subway_route_map.py       # 第 1 步：爬线路图+规划图
python scrapers/scrape_metrodb.py       # 第 2 步：爬客流数据
python scrapers/generate_charts.py      # 第 3 步：生成图表
```

- 数据仅供个人学习研究使用，严禁商业用途
- 建议控制抓取频率，避免对源站造成压力（并发 8 已是合理上限）
